# Dashboard（QClaw 风格 renderer）日本語化

## 方針

- **意味を理解した上で自然な日本語に**。中文の直訳は避ける。
- **画面上は日本語のみ**。中国語・英語の UI 表記は残さない。

## 置換一覧（dashboard-renderer）

| 元（QClaw 中文） | 日本語（意訳・自然表現） |
|------------------|--------------------------|
| 技能广场 | スキル広場 |
| 搜索 | 検索 |
| 今天 / 本周 / 本月 | 今日 / 今週 / 今月 |
| 微信远程 | リモート接続（「微信」は固有名のため汎用表現に） |
| 用户系统配置 | ユーザー・システム設定 |
| 检测更新 | 更新を確認 |
| 可以描述任务或提问任何问题 | タスクや質問を入力してください |
| 内容由AI生成，请仔细甄别 | AIによる生成内容です。内容の正確性にご注意ください。 |
| 已截断 (${n} 字符…) | 省略（全${n}文字のうち、先頭${m}文字を表示） |
| 请求失败 | リクエストに失敗しました |

## 変更箇所

- **dashboard-renderer/index.html**  
  - `lang="zh-CN"` → `lang="ja"`  
  - コメント「微信登录 SDK」→ 英語 "(optional)"（開発者向け）
- **dashboard-renderer/assets/c-8aHI0XC0.js**  
  - 上記文案をすべて置換（replace_all）。

## 一括置換スクリプト

- **scripts/dashboard-zh-to-ja.js**  
  - `dashboard-renderer/assets/*.js` 内の中国語 UI 文案を一括で日本語に置換。  
  - 実行: `node japanclaw-setup/scripts/dashboard-zh-to-ja.js`  
  - マッピングは 300 件以上（初期化・エラー・ボタン・モデル・会話・微信・招待碼など）。  
  - QClaw 側を再取り込みした場合は本スクリプトを再実行してからビルドすること。

## 注意

- 他チャンク（e-BOOjGo6C.js 等）の英文は Zod 等ライブラリのエラーメッセージで、画面上に出ない想定のため未変更。
- **control-ui-ja** は別 UI（OpenClaw 公式風）。zh-CN/zh-TW ロケールは言語切替用に残している。デフォルトを日本語にする場合は設定で locale=ja を指定する。
